前情提要：
如何声明函数？如何调用函数?
function 函数名（形参）{
    函数体
    return 返回值
}
函数名(实参列表)
1.函数命名规则 及基本特性？
数字下划线 字母，非数字开头，不区分大小
不能重复定义
在任意处调用

2.函数的参数分为哪两种？各有什么特点？
形参
    1.形式上参
    2.可以有默认值
    3.右
实参
    1.实际参 -- 调用
    2.多余会被省略
    3.按照形参来传递

3.什么是返回值？特性是？
return 定义返回值
null 
return 之后不执行

4.return 与 echo的区别？
使用return ,是函数执行完毕后，返回回来的结果（调用处）
使用echo ,是函数在执行过程中，直接将信息输出到页面，什么都没有返回给你。

5.什么是局部变量及特性，如何在外部使用局部变量？
在函数内部声明的变量就是局部变量
函数执行完毕之后，局部变量会被释放掉
global 关键字
