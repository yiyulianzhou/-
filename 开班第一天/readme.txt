	在web中html/css/php都是做什么的:
	大家都用过QQ空间吧,你打开QQ空间后网页展示给你的内容就是html去实现的.

	然后你发现不同的等级可以搭配不同的模版组件(黄钻绿钻等等),这些是由css来改变的.

	那么就剩下php了,它能做什么.你想一下,你有没有登录,网页需要有判断吧? 登陆后每个人的好友关系都不一样吧,每个人的个人资料照片日志也都不一样吧.

	这些都是php来实现的,当你登陆时就向服务器发送请求,服务器端的php验证通过后(哦,自己人),你就登录成功了.登录成功以后,php还要把你的个人资料从数据库拿出来,把你的照片日志等等东西拿出来转为相应的文件输出到页面.

	所以,现在你发现php是用来:接收和处理数据的,负责查询和传递数据的.

	一句话概括php就是:服务器上的搬运工
	服务器这个东西你想一下,它是不是在很远的地方,也许是杭州/北京/加尼福尼亚/放着的一台机器,你的数据都在这么远的地方放着.当你向服务器拿某些数据时,服务器可听不懂你丫到底要干什么(它只想静静的做一台机器)

	此时php就想要干活了,php就像一条狗一样:你要数据是吧,好滴.然后他在服务器或数据库(数据库也是服务器)上找到相应的数据,并转化成客户端能解析的html/css等文件,然后通过服务器给你发送回来.

	当你登陆时,点击链接时,远在天边的服务器上的php都是这么工作的

		什么是会话控制,简单说一下,web页面是通过http协议传输实现的,HTTP也叫做无状态协议.

		就是当你请求一个页面,它就做出响应返回给你一个页面,请求+响应就是一个会话.

3.编辑器
		千万不要使用Windows自带的记事本编辑任何文本文件.原因是Microsoft开发记事本的团队使用了一个非常弱智的行为来保存UTF-8编码的文件,他们自作聪明地在每个文件开头添加了0xefbbbf(十六进制)的字符,你会遇到很多不可思议的问题,比如,网页第一行可能会显示一个“?”,明明正确的程序一编译就报语法错误,等等,都是由记事本的弱智行为带来的.
		编辑器不做强制要求,只要顺手就行.
但是有一个要求,不管用什么编辑器,在初级项目前,把单词提示和自动补全这些功能或者插件都关掉.
关掉这些是为了让你们在敲代码时能记住属性/单词/代码格式.
你若能记住这些东西,不关掉也可以,但我会在课上提问,项目经理会做课后测试,凡是没关提示补全的,罚抄比别人多一倍.注意是手抄.
